In this project we develop some tools and mantain knowledge bases related to confiscated assets. The goal is to build a huge knowledge base of confiscated assets in Sicily: a generic vocabulary to represent confiscated assets, and ontology of which public administrations provide enough information about confiscated assets it owns, ontologies of these assets. We release the tools used to build this knowledge base as well, in particular tools to convert assets information in its origin format (may be, for example, CSV) in the uniform format we provide.
